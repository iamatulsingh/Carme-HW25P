I've tested this code on Linux (Ubuntu 18.04). I don't know whether it will work on Mac or Windows.

### How to install required dependecies?
`pip3 install -r requirements.txt`

### How to execute code?
`python3 example.py -m xx:xx:xx:xx:xx:xx`
